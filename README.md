# 3차 프로젝트
### 👋&nbsp; 1.소개
   - 비트캠프 3차 프로젝트 진행(2021-11-02 ~ 2021-11-23(22))
   - 프로젝트 명 : 미룸
   - 프로젝트 인원 : 6명
   - 프로젝트 선정이유 : 제주도로 국내 여행을 떠난 여행객들을 위해서 제주도를 기반으로 한 기업들과 상생해서 프로모션과 이벤트를 제공하며,<br>
   여행을 편하게 즐길 수 있는 숙소와 액티비티와 원데이 클래스로 제주도의 문화와 추억을 쌓을 수 있게 해주는 서비스의 사이트
